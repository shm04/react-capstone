# 📖 React Capstone Project <a name="about-project"></a>


**React Capstone Project** is a SPA web app that works with real live data from a football soccer API. The application will allow users to find information on leagues worlwide.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

Text Editor 

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/shm04/react-capstone.git
```

### Install

Install this project with:


```sh
  cd my-folder
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```


### Run tests

To run tests, run the following command:

```sh
  npm test 
```
